SYNOPSIS

       #include <liburing.h>

       int io_uring_submit_and_wait(struct io_uring *ring,
                                    unsigned wait_nr);

DESCRIPTION

       The io_uring_submit_and_wait(3) function submits the next requests from
       the submission queue belonging to the ring and waits for wait_nr
       completion events.

       After the caller retrieves a submission queue entry (SQE) with
       io_uring_get_sqe(3) and prepares the SQE, it can be submitted with
       io_uring_submit_and_wait(3).

RETURN VALUE

       On success io_uring_submit_and_wait(3) returns the number of submitted
       submission queue entries. On failure it returns -errno.
