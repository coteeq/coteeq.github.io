SYNOPSIS

       #include <liburing.h>

       void io_uring_buf_ring_init(struct io_uring_buf_ring *br);

DESCRIPTION

       io_uring_buf_ring_init(3) initialises br so that it is ready to be used.
       It may be called after io_uring_register_buf_ring(3) but must be called
       before the buffer ring is used in any other way.

RETURN VALUE

       None
