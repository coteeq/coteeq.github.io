SYNOPSIS

       #include <sys/mman.h>
       #include <liburing.h>

       void io_uring_prep_madvise(struct io_uring_sqe *sqe,
                                  void *addr,
                                  off_t len,
                                  int advice);

DESCRIPTION

       The io_uring_prep_madvise(3) function prepares an madvise request. The
       submission queue entry sqe is setup to start an madvise operation at the
       virtual address of addr and of len length in bytes, giving it the advise
       located in advice.

       This function prepares an async madvise(2) request. See that man page for
       details.

RETURN VALUE

       None

ERRORS

       The CQE res field will contain the result of the operation. See the
       related man page for details on possible values. Note that where
       synchronous system calls will return -1 on failure and set errno to the
       actual error value, io_uring never uses errno.  Instead it returns the
       negated errno directly in the CQE res field.

SEE ALSO

       io_uring_get_sqe(3), io_uring_submit(3), io_uring_register(2), madvise(2)
